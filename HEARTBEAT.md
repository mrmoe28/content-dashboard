# HEARTBEAT.md - Periodic Check List

*Quick checks to run during heartbeat polls. Rotate through these, 2-4 times per day.*

## Quick Checks (5-10 min total)

### Email
- [ ] Any urgent unread messages?
- [ ] Action items from recent messages?

### Calendar
- [ ] Upcoming events in next 24-48 hours?
- [ ] Any conflicts or prep needed?

### Mentions / Social
- [ ] Discord/Slack mentions?
- [ ] Important notifications?

### System Health
- [ ] LM Studio running smoothly?
- [ ] Ollama responsive?
- [ ] Gateway/browser active?

## Conditions

**When to reach out proactively:**
- Important email arrived
- Calendar event coming up (<2h)
- Something interesting discovered
- It's been >8h since last contact

**When to reply HEARTBEAT_OK:**
- Late night (23:00-08:00) unless urgent
- Nothing new to report
- Human is clearly busy
- Checked <30 minutes ago

## Background Tasks (Can do quietly)

- Update daily memory file
- Review and commit any workspace changes
- Organize session logs
- Check project git status
